# CUMTBeamer
这是一个中国矿业大学Beamer模版
## 模版主题:
使用了Madrid&&Beaver
加入了中国矿业大学logo，右上角logo也包含在代码中，有需要的可以自行取消注释添加
## 模版包含了以下内容:
1.文字
2.图片
3.代码
4.表格
5.公式
## 模版使用方法:
1.下载模版
2.将tex文件中的内容替换为自己的内容
3.将code和fig文件夹中的内容替换为自己的内容
4.编译
